Para ejecutar el programa se necesita poner la siguiente linea de comando en terminal:
gcc nombre_del_archivo.c -fopenmp
Una vez que que se corrió se generará un archivo a.out el cual se correrá para ejecutar el programa de la siguiente manera:
./a.out
Por último la manera de usar el programa es ingresar el número de numeros que posteriormente otorgará, dar enter y después ingresar los numeros, deben ser la misma cantidad de numeros que el número que se mando al incio del programa
El programa solo imprimira los números que se ingresaron en orden
